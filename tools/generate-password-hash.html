Generate a new site password

This page runs entirely in your browser — nothing is sent anywhere. Type a new password, copy the hash it produces, and paste it into assets/js/config.js as SITE_PASSWORD_HASH.

Where to paste it

Open assets/js/config.js and replace the value like this:

SITE_PASSWORD_HASH: "paste-your-new-hash-here",

Save the file, then commit and push it to GitHub for the live site to use the new password.
